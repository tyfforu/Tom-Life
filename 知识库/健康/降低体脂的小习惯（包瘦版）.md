# 降低体脂的小习惯（包瘦版）

> 来源：小红书

## 核心原则

**稳 > 快**——别追求瘦得快，身体适应了才是真瘦，稳一点才不反弹。

---

## 饮食习惯

### 吃什么

| 类型 | 建议 |
|------|------|
| ✅ 优质蛋白 | 鸡胸肉、鱼肉、豆腐、鸡蛋——长瘦肉的底牌，放心吃 |
| ✅ 蔬菜 | 随便吃，管饱不胖（前提：少油少酱料） |
| ✅ 碳水 | 适量吃，别彻底断，否则头发先掉 |
| ✅ 蛋白质饮品 | 每天保证一杯，饱腹感强，不容易乱吃 |
| ✅ 黑咖啡/抹茶 | 提神 + 促代谢，可以多喝 |
| ❌ 动物皮 | 全是油，吃一口≈跑十分钟 |
| ❌ 奶茶/含糖饮料 | 隐形热量炸弹，能戒就戒，戒不掉就少糖少料 |
| ❌ 水果当饭吃 | 果糖比想象中狠，吃多了照样囤肉 |
| ❌ 酱料 | 沙拉酱、花生酱、辣油热量惊人，蘸一下≈半碗饭 |

### 怎么吃

- **进食顺序**：水 → 蔬菜 → 肉 → 碳水（血糖稳，热量摄入也少）
- **吃慢点**：饱的信号传到大脑需要时间，吃太快容易超标
- **烹饪方式**：蒸煮焖炖 > 煎炸炒，少一半热量
- **饭前先喝水**：饿了先喝水，很多时候不是真饿，是渴了

---

## 运动 & 生活习惯

- 🚶 **饭后别马上坐下/躺下**：站十分钟或慢走一会儿，脂肪不容易堆肚子
- 🪑 **上班别久坐**：屁股会塌、大腿会粗，站起来走走就赢了
- 🏃 **把活动拆进生活**：能站着就别坐，能走楼梯就别坐电梯
- 😴 **保证睡眠**：睡得少真的瘦得慢，多睡觉是懒人减脂捷径
- 🌙 **睡前保持一点饥饿感**：身体在悄悄燃脂的信号
- ⚖️ **定期上秤**：数字会提醒你管住嘴
