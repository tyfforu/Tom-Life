### **一、基础操作（必考）**
#### 1. **文件与目录管理**
| 命令 | 作用 | 考点示例 |
|------|------|----------|
| `ls -l /path` | 列出目录内容（长格式） | 查看 `/var/log` 中文件的权限、大小、修改时间 |
| `cd /path` | 切换目录 | 从家目录进入 `/etc/sysconfig` |
| `mkdir -p /a/b/c` | 递归创建多级目录 | 创建路径 `/data/project/logs` |
| `rm -rf /path` | 强制删除目录（**危险操作！**） | 删除临时文件夹 `/tmp/test`（注意权限） |
| `cp -r /src /dest` | 递归复制目录 | 将 `/etc` 下的 `ssh` 文件夹复制到 `/backup` |

**避坑提示**：  
- 初赛常考 **`-r`（递归）** 和 **`-f`（强制）** 参数的使用场景，例如删除目录必须加 `-r`，否则报错。  
- `rm -rf /` 是致命错误，考试中若出现此类命令需直接判断为错误操作。

---

#### 2. **文件内容查看**
| 命令 | 作用 | 考点示例 |
|------|------|----------|
| `cat file` | 查看文件全部内容 | 查看 `/etc/hosts` 配置 |
| `less file` | 分页查看大文件（**推荐考试用**） | 查看日志文件 `/var/log/messages` |
| `head -n 10 file` | 查看前 10 行 | 查看配置文件头部注释 |
| `tail -f /log` | 实时跟踪日志更新 | 监控 Web 服务日志 `/var/log/nginx/access.log` |

**避坑提示**：  
- 大文件必须用 `less` 或 `tail`，直接 `cat` 可能卡死终端（考试环境会扣分）。

### **二、文本处理（高频考点）**
#### 1. **搜索与过滤**
| 命令 | 作用 | 考点示例 |
|------|------|----------|
| `grep "error" /log` | 搜索包含 "error" 的行 | 从日志中筛选错误信息 |
| `grep -i "ERROR" /log` | 忽略大小写搜索 | 搜索日志中的 "ERROR" 或 "error" |
| `grep -v "success" /log` | 排除包含 "success" 的行 | 筛选出非成功的操作记录 |

#### 2. **管道符（`|`）组合命令**
```bash
# 示例1：统计当前目录下 .conf 文件数量
ls | grep ".conf" | wc -l

# 示例2：查看内存使用率最高的 5 个进程
ps aux | sort -k4nr | head -5

# 示例3：提取 IP 地址（考试高频）
cat /var/log/secure | grep "Failed" | awk '{print $11}' | sort | uniq -c
```

**避坑提示**：  
- 管道符是**考试必考操作**，需掌握 `grep` + `awk` + `sort` 的组合逻辑。  
- 注意 `awk` 默认以空格分隔字段，`$11` 表示第 11 列（如日志中 IP 通常在第 11 位）。

### **三、权限与环境变量（易错点）**
#### 1. **权限管理**
| 命令 | 作用 | 考点示例 |
|------|------|----------|
| `chmod 755 file` | 修改权限（r=4, w=2, x=1） | 使脚本 `run.sh` 可执行 |
| `chown user:group file` | 修改所有者 | 将 `/data/app` 所有者改为 `nginx:nginx` |

**避坑提示**：  
- 初赛必考 **数字权限计算**：  
  - `755` = `rwxr-xr-x`（所有者全权限，组和其他人可读可执行）  
  - `644` = `rw-r--r--`（所有者可读写，组和其他人只读）

#### 2. **环境变量**
| 命令 | 作用 | 考点示例 |
|------|------|----------|
| `echo $PATH` | 查看 PATH 变量 | 确认命令搜索路径 |
| `export VAR="value"` | 临时设置环境变量 | 使变量在当前会话生效 |
| `vim ~/.bashrc` | 永久设置环境变量 | 添加自定义命令路径 |

**避坑提示**：  
- 考试中若需**永久生效**，必须编辑 `~/.bashrc` 或 `/etc/profile`，直接 `export` 仅临时有效。

### **四、Shell 脚本基础（初赛入门级）**
#### 1. **简单脚本结构**
```bash
#!/bin/bash
# 示例：批量创建用户
for user in user1 user2 user3; do
  useradd $user && echo "$user created" || echo "$user exists"
done
```
- **考点**：  
  - `#!/bin/bash`（必须写在第一行）  
  - 循环结构 `for...do...done`  
  - 逻辑判断 `&&`（成功执行）和 `||`（失败执行）

#### 2. **变量与参数**
```bash
name="OpenEuler"
echo "Hello, $name"   # 输出：Hello, OpenEuler
```
- **考点**：  
  - 变量赋值**不能有空格**（`name = "value"` 错误，`name="value"` 正确）  
  - 引用变量需加 `$`（`$name`）

### **五、考试高频题型 & 答题技巧**
#### 1. **场景题：给出问题，写出命令**
> **题目**：从 `/var/log/secure` 中找出所有 SSH 登录失败的 IP 地址，并统计出现次数。  
> **答案**：  
> ```bash
> grep "Failed password" /var/log/secure | awk '{print $11}' | sort | uniq -c
> ```

#### 2. **判断题：命令是否正确**
> **题目**：`rm -f /etc/*.conf` 会删除 `/etc` 下所有 `.conf` 文件。  
> **答案**：✅（`-f` 强制删除，`*.conf` 通配符匹配）

#### 3. **填空题：补全命令**
> **题目**：查看进程 `nginx` 的 PID，命令为 `____ -C nginx`。  
> **答案**：`pgrep`

### **六、关键总结（考试必背）**
| 类别 | 必考命令 | 口诀 |
|------|----------|------|
| 文件操作 | `ls`, `cp`, `rm`, `mkdir` | `-r` 递归，`-f` 强制 |
| 文本处理 | `grep`, `awk`, `管道符` | `grep` 找行，`awk` 取列 |
| 权限管理 | `chmod 755`, `chown` | `r=4, w=2, x=1` |
| 脚本基础 | `#!/bin/bash`, `for`, `$var` | 变量无空格，循环用 `do` |

**最后提醒**：  
- 初赛操作题需**手敲命令**，务必熟悉键盘位置（如 `|` 在 `Enter` 上方）。  
- 所有命令**区分大小写**（如 `grep "Error"` 和 `grep "error"` 结果不同）。  
- 遇到不确定的命令，用 `man 命令` 或 `--help` 查看帮助（考试环境允许）。
