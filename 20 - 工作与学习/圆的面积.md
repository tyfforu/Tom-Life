这是一个非常适合初学者练习QT（Qt Widgets Application）的小项目。我将把过程分解为**5个核心步骤**，确保你照着做就能运行成功。

**前提条件**：你已经安装好了 Qt Creator 和相应的编译器（MinGW 或 MSVC）。

---

### 第一步：创建新项目

1.  打开 **Qt Creator**。
2.  点击 **Create Project** (创建项目)。
3.  选择 **Application (Qt)** -> **Qt Widgets Application**，点击 **Choose**。
4.  **项目名称 (Name)**：输入 `CircleArea`（或者你喜欢的名字）。
5.  **构建系统 (Build System)**：默认选择 `qmake` 即可。
6.  **类信息 (Class Information)**：
    *   Class name: `MainWindow` (保持默认)
    *   Base class: `QMainWindow` (保持默认)
7.  **Kit Selection**：选择你电脑上安装的编译器（通常会有默认选中的），一路点击 **Next** 直到 **Finish**。

---

### 第二步：设计界面 (UI)

我们要画出“输入框”、“按钮”和“显示结果的标签”。

1.  在左侧项目文件列表中，双击打开 **`Forms`** -> **`mainwindow.ui`**。这会进入图形化设计界面。
2.  从左侧的 **Widget Box** (控件箱) 中拖拽以下控件到中间的窗口上：
    *   **Label (标签)** x 2：
        *   第一个 Label 双击修改文本为：“请输入半径：”
        *   第二个 Label 双击修改文本为：“面积将在这里显示”
    *   **Line Edit (单行输入框)** x 1：用来输入半径数字。
    *   **Push Button (按钮)** x 1：双击修改文本为“计算面积”。

3.  **关键步骤：修改对象名称 (Object Name)**
    为了在代码里容易找到这些控件，我们需要给它们改个英文名（在右侧的属性栏 **Property** 中修改 `objectName`）：
    *   点击那个**输入半径的输入框**，将 `objectName` 改为 `le_radius`。
    *   点击那个**显示结果的标签**，将 `objectName` 改为 `lbl_result`。
    *   点击那个**计算按钮**，将 `objectName` 改为 `btn_calculate`。

4.  （可选）为了好看，你可以把它们简单排列一下，或者使用上方的布局工具（水平/垂直布局）。

---

### 第三步：编写逻辑代码 (关联信号与槽)

我们要告诉程序：“当按钮被点击时，执行计算代码”。

1.  在 UI 设计界面中，**右键点击** 你刚才放好的“计算面积”按钮。
2.  选择 **Go to slot... (转到槽...)**。
3.  在弹出的窗口中选择 **`clicked()`**，然后点击 OK。
4.  Qt Creator 会自动跳转到 `mainwindow.cpp` 文件，并为你生成好了一个函数的空壳：
    ```cpp
    void MainWindow::on_btn_calculate_clicked()
    {
    }
    ```

---

### 第四步：实现计算公式

现在在刚才生成的函数大括号 `{ }` 里写 C++ 代码。

请将 `mainwindow.cpp` 修改为如下内容（重点看 `on_btn_calculate_clicked` 函数部分）：

```cpp
#include "mainwindow.h"
#include "ui_mainwindow.h"
#include <cmath> // 引入数学库，为了使用圆周率 M_PI

MainWindow::MainWindow(QWidget *parent)
    : QMainWindow(parent)
    , ui(new Ui::MainWindow)
{
    ui->setupUi(this);
    // 设置窗口标题
    this->setWindowTitle("圆面积计算器");
}

MainWindow::~MainWindow()
{
    delete ui;
}

// 这是我们要写的核心代码
void MainWindow::on_btn_calculate_clicked()
{
    // 1. 获取输入框里的文本
    QString strRadius = ui->le_radius->text();

    // 2. 将字符串转换为 double (浮点数)
    bool ok;
    double radius = strRadius.toDouble(&ok);

    // 3. 判断用户输入的是不是数字
    if(ok) {
        // 如果转换成功，计算面积： Area = PI * r * r
        // M_PI 是 cmath 库里定义的π，约等于 3.14159...
        double area = M_PI * radius * radius;

        // 4. 将计算结果转回字符串，并保留2位小数
        QString strArea = QString::number(area, 'f', 2);

        // 5. 显示在结果标签上
        ui->lbl_result->setText("圆的面积是：" + strArea);
    } else {
        // 如果输入的不是数字
        ui->lbl_result->setText("错误：请输入有效的数字！");
    }
}
```

---

### 第五步：运行程序

1.  点击左下角的 **绿色三角形按钮 (Run)**，或者按快捷键 `Ctrl + R`。
2.  程序编译完成后会弹出一个窗口。
3.  在输入框输入一个数字（例如 `10`），点击“计算面积”。
4.  结果标签应该会显示：`圆的面积是：314.16`。

---

### 总结代码逻辑

整个过程的核心逻辑就是三句话：
1.  **Input**: `ui->le_radius->text()` (从界面拿数据)
2.  **Process**: `Area = π * r²` (在后台算数据)
3.  **Output**: `ui->lbl_result->setText(...)` (把结果放回界面)

照着这个步骤，你就能完成你的第一个QT计算程序了！
