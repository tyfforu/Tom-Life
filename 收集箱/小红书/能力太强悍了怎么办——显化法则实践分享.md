# 能力太强悍了怎么办——显化法则实践分享

> 来源：小红书 @猫猫头

## 核心内容

作者分享了自己通过**显化法则**（吸引力法则）在生活中"实现"的多件事，以下是具体案例：

| 序号 | 事项 | 结果 |
|------|------|------|
| ① | **体重管理** | 吃不吸收，一天拉两次，肚子越来越扁平 |
| ② | **屏蔽广告** | 闭眼想了一下，翻了几十页都没有广告出现 |
| ③ | **恋人关系** | sp 每时每刻粘着自己，24 小时待在一起 |
| ④ | **身体设定** | 给自己"鲜花"了虚弱体质，中医把脉说很虚但很健康 |
| ⑤ | **容貌改变** | 打扮后同学认不出来，被喊"卧槽"，被夸女神、漂亮 |
| ⑥ | **美甲颜色** | 美甲店没有的颜色都被调出来，每次都很满意 |
| ⑦ | **肤色变白** | 越来越白，甚至越晒太阳反而越白 |
| ⑧ | **影响他人** | 对固定 NPC 做实验：被点名、摔跤、穿特定颜色衣服 |
| ⑨ | **金钱显化** | 大学期间月入 1w+，对现实运筹帷幄 |

## 关键词

- **显化法则**：通过意念/想象力改变现实的信念体系
- **鲜花（正向显化）**：给自己或他人设定正面意图
- **NPC**：作者将周围人视为"游戏中的 NPC"并进行意念实验
