# 收集箱

> 临时存放新笔记，等待整理分类

## 使用规则

1. **新笔记默认放入此处**，不直接分类
2. **按来源分子文件夹**：
   - `小红书/` - 小红书帖子
   - `网页剪藏/` - 浏览器剪藏内容
   - `待整理/` - 其他来源
3. **发出「整理」指令后**，再按分类规则归档到正式目录

## 当前分类体系

| 一级目录 | 子分类 | 存放内容 |
|----------|--------|----------|
| **00 - 记录** | 日记、习惯打卡 | 个人日常记录 |
| **10 - 人员管理** | - | 人际关系、联系人信息 |
| **20 - 工作与学习** | 嵌入式IDE、学习方法、英语、自媒体、AI工具 | 职业技能、学习资料 |

| **40 - Obsidian** | 模板 | Obsidian 配置、模板 |
| **卡片知识** | - | 原子化知识卡片 |
| **Clippings** | - | 已整理的剪藏内容（将逐步迁移） |

## 整理指令

当你说以下任意一种话时，我会帮你整理收集箱中的笔记：
- "整理笔记"
- "分类"
- "归档"
- "把收集箱的笔记整理一下"
