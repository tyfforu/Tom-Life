# 怎样避免习惯性吃撑（干货版）

> 来源：小红书

## 核心技巧总结

| 序号 | 方法 | 操作要点 |
|------|------|----------|
| ① | **饱嗝信号法** | 打出第一个饱嗝时，差不多是七八分饱，此时应逐渐停下来 |
| ② | **摸胃判断法** | 吸气后摸胃部：微微凹→七分饱以下；平着→七八分饱（别吃了）；吸气还突出→十二分饱 |
| ③ | **细嚼慢咽法** | 每口饭嚼15-20次，控制吃饭时间在20分钟左右，吃平时80%的饭量就会饱 |
| ④ | **补充镁和B族** | 缺乏时神经系统易兴奋，对甜食碳水渴望增强。多吃深绿蔬菜、坚果、全谷物、豆类 |
| ⑤ | **餐前温水法** | 餐前喝200ml 40℃温水，让胃有充盈感。可加苹果醋让食欲更平静 |
| ⑥ | **避免糖油混合** | 高糖+高脂（如糖醋排骨、红烧肉配饭、甜甜圈）是大脑最难抗拒的组合，尽量分开吃 |
| ⑦ | **区分饿与馋** | 饿=胃空+手脚无力；馋=嘴巴无聊/情绪低落。馋时给自己10分钟缓冲，转移注意力 |
| ⑧ | **别过度节食** | 平时吃太少会导致食欲反扑，正常吃三餐、七八分饱，身体不缺营养就不会暴食 |
| ⑨ | **薄荷糖收尾** | 饭后想吃点什么时，含无糖薄荷糖或嚼口香糖，清凉感抑制食欲 |

## 重点原理

### 为什么细嚼慢咽有效？
咀嚼不只是把食物嚼碎，而是会调动整个消化系统，包括传送饱腹感给大脑。坚持一星期左右就会形成肌肉记忆。

### 为什么糖油混合物危险？
甜咸搭配、糖油混合物同时刺激大脑的多个奖励通路，让你在完全不饿的情况下也能一直吃。

### 10分钟缓冲法
馋的时候承诺自己：「如果10分钟之后还想吃，我就立刻吃。」这10分钟可以：
- 出门丢个垃圾
- 整理房间
- 做一点运动
