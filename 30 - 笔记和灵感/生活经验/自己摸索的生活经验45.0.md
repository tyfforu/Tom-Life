## 1. 设置你的 Siri！

> [!tip] 核心思路
> 来源于不安 — 手机静音 + 容易丢东西 + 流量卡接不了电话

### 为什么设置

- 手机经常设置成静音
- 容易丢东西，手机不见了就真的相当于没了
- 电话卡是流量卡，打电话也接不了

### 实际用途

| 场景 | 指令 |
|------|------|
| 手机丢了 | 满屋子乱走呼叫 "嘿 Siri" |
| 两手提东西 | "嘿 Siri，打开手电筒" |
| 不方便摸手机 | "嘿 Siri，播放音乐" |

## 2. 生病小经验

> [!warning] 健康红线
> 熬夜、吹凉风、饮食不规律 **这三样不能同时出现**

### 生病时

- [ ] 吃多点有营养的食物
- [ ] 保证充足睡眠
- [ ] 保持环境干净又温暖

### 恢复期

> [!danger] 重要
> 刚从小病缓过来，**千万不要熬夜**，很有可能让病毒打赢复活赛！

### 身体信号

如果身体越来越不舒服，**立刻上床休息睡觉**，而不是爬床上继续玩手机。

**身体发出的信号不要忽略。**

## 3. 口腔+舌头溃疡小经验

> [!success] 秘方：蓝莓 + 溃疡膏搭配使用

### 治疗方法

1. **吃蓝莓** — 蓝莓有抗炎功效
2. **漱口**
3. **涂药膏**

### 涂药膏 Tips

| 要点 | 说明 |
|------|------|
| 工具 | 直接拿手指涂，更准、面积更大 |
| 卫生 | 每次用没涂抹过的手指，避免二次污染 |
| 手法 | 尽量不要来回抹 |
| 忌口 | 涂完后30分钟内不要吃喝 |

> [!note] 频率
> 一天内多次涂抹效果更佳

### 效果

- 药膏：日本买的，尝起来甜甜的不排斥
- 周期：**涂了两天就差不多好了**
- 之前状态：疼得说不出话，吃东西都痛

## 5. 逼迫自己学习的邪修方法

eyman和stereo

> [!example] 核心机制
> 用真金白银倒逼学习动力

### 操作方法

1. **看到博主推荐的基金** → 买 **10块钱** 玩玩（最低投资就是10元）
2. **觉得有道理** → 查资料、研究
3. **验证后** → 追加投资

### 实际效果

- 引导学习到了很多以前不会接触的东西
- 打开了新世界的大门

> [!warning] 重要提醒
> 投资有风险，建议大家一定要谨慎，不当韭菜！
> 
> **要把他作为一个引导学习的手段，而不是花钱的手段。**

### 核心理念

```
学习 → 得到认知 → 认知带来财富
```
