# 怎么做读书笔记

> 来源：VICKY学姐
> 古训：读书要做到「眼到、口到、心到、**手到**」——手到就是读书笔记。
> 最淡的墨水，也胜过最强的记忆。——托尔斯泰

---

## 为什么写读书笔记

1. **深入理解，牢固掌握**所学知识
2. **积累资料**，以备不时之需
3. **提高科学研究和写作能力**
4. 写笔记的过程本身就是**训练阅读**的好方法

---

## 读书笔记三大类型

### 类型一：摘要式（积累资料）

把原文关键内容原封不动抄下来，注明出处（题目、作者、出版单位、日期、页码）。

**两种形式：**
- **索引笔记**：只记出处页码，不抄内容（适合量大时快速建档）
- **抄录原文**：照抄精彩语句、段落、数据（适合日后引用）

> 摘抄内容要筛选，以是否有用为标准。

---

### 类型二：评注式（理解加工）

不只是摘录，还要写出自己的观点和看法。

| 形式 | 做法 |
|------|------|
| **书头批注** | 在书页空白处打符号、加批注、折页、夹纸条 |
| **提纲** | 按章节逐段写出要点，可用原文语句 + 自己语言 |
| **提要** | 综合全文写要点，完全用自己的语言，带评述性质 |
| **评注** | 读完对得失加以评论，或对疑难之点加以注释 |
| **补充原文** | 围绕中心思想引申或发挥，不是随意补充 |

---

### 类型三：心得式（产出观点）

读书后写出认识、感想、体会和启发，是最高阶的笔记形式。

| 形式 | 做法 |
|------|------|
| **札记** | 摘记要点 + 心得结合，形式灵活，可长可短 |
| **心得（读后感）** | 以自己语言为主，写体会、感想、收获，可适当引用原文 |
| **综合笔记** | 读多本同题书文后，评论各方观点，提出自己看法 |

---

## 笔记形式选择

| 形式 | 优点 | 缺点 |
|------|------|------|
| 成册笔记本 | 便于保存 | 不便分类 |
| 活页本 | 便于分类、节约纸张 | — |
| 卡片 | 便于分类排列、灵活调动 | 篇幅小，不宜写长内容 |
| 剪报 | 保留完整，来源真实 | 整理耗时 |
| 全文复印 | 保持完整性 | 占用空间 |
| 记忆 | 即取即用 | 依赖记忆力 |

---

## 笔记能力进阶路径

```
摘要式（入门）→ 评注式（进阶）→ 心得式（高阶·科学研究）
```

开始时用摘要式，读书多了有比较、产生了看法，再写评注式。心得式属于科学研究范畴，是更进一步的产出。
